# tracker
## 🛠 Стек технологий

В проекте используется следующий стек технологий:

### Основной фреймворк
- **[Next.js](https://nextjs.org/)** (App Router) — React-фреймворк для продакшн-приложений (поддержка фронтенда и бэкенда).
- **[React](https://reactjs.org/)** — библиотека для построения пользовательских интерфейсов.
- **[TypeScript](https://www.typescriptlang.org/)** — типизированная надстройка над JavaScript для надежности и удобства разработки.

### Структура проекта
- `app/` — роутинг, макеты и страницы приложения (Next.js App Router).
- `components/` — переиспользуемые UI-компоненты.
- `hooks/` — пользовательские хуки React для общей логики.
- `lib/` — утилиты, хелперы и конфигурации сторонних сервисов.
- `scripts/` — кастомные Node.js скрипты для автоматизации.
- `styles/` — глобальные стили и CSS-модули.

### База данных
- **[PostgreSQL](https://www.postgresql.org/)** — объектно-реляционная база данных с открытым исходным кодом.
